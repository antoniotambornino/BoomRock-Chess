BoomRock Chess is a Progressive Web App (PWA) — install it directly from any browser on iOS, Android, macOS, or Windows 10/11. No app store account required. The app runs offline with low-latency cached responses.

Install directly from your browser — no account or store needed:

🧭 Safari — iPhone & iPad iOS
  1. Open this page in Safari (required — Chrome/Firefox on iOS use Safari's engine but may lack PWA install)
  2. Tap the Share ⎙ button at the bottom of the screen
  3. Scroll down the share sheet and tap "Add to Home Screen"
  4. Name it BoomRock Chess and tap Add
  5. The icon appears on your home screen — launches full-screen like a native app ✓
🌐 Chrome — Android Android
  1. Open this page in Google Chrome
  2. Look for the install icon ⊕ in the address bar and tap it, or
  3. Tap the ⋮ three-dot menu → "Add to Home screen" / "Install app"
  4. Tap Install — BoomRock Chess runs in its own window ✓
🦊 Firefox — Android Android
  1. Open this page in Firefox for Android
  2. Tap the ⋮ three-dot menu in the top-right corner
  3. Tap "Install" or "Add to Home Screen"
  4. Confirm — BoomRock Chess installs as a standalone app ✓
🪟 Microsoft Edge — Windows 10 / 11 Windows
  1. Open this page in Microsoft Edge (built into Windows 10/11)
  2. Look for the ⊕ app install icon in the address bar — click it, or
  3. Click the ⋯ three-dot menu → Apps → Install this site as an app
  4. Click Install — BoomRock Chess opens in its own app window and appears in your Start menu ✓
🌐 Chrome — Windows 10 / 11 Windows
  1. Open this page in Google Chrome
  2. Click the ⊕ install icon in the address bar (right side), or
  3. Click the ⋮ three-dot menu → "Cast, save and share" → Install page as app…
  4. Click Install — BoomRock Chess is pinned to your taskbar and Start menu ✓
🧭 Safari — Mac macOS Sonoma+
  1. Open this page in Safari on macOS 14 Sonoma or later
  2. In the menu bar choose File → Add to Dock…
  3. Click Add — BoomRock Chess is pinned to your Dock as a standalone app ✓
🌐 Chrome — Mac macOS
  1. Open this page in Google Chrome on Mac
  2. Click the ⊕ install icon in the address bar, or
  3. Click ⋮ menu → "Cast, save and share" → Install page as app…
  4. Click Install — BoomRock Chess opens in its own window and appears in Launchpad ✓

💚 FREE — 988 Suicide & Crisis Lifeline

The 988 Lifeline is a free, confidential service available 24 hours a day, 7 days a week. You can call or text 988 to reach a trained crisis counselor. No insurance. No cost. No judgment.

🌐 Visit 988lifeline.org 📞 Call or Text 988

Services available in English and Spanish. Chat also available at 988lifeline.org. Veterans can press 1 after dialing for the Veterans Crisis Line.

💳 Professional Therapy — Requires Payment

Licensed therapists provide ongoing, personalised mental health care. Sessions are not free — you pay out of pocket, through health insurance, or via a sliding-scale fee based on income. Typical costs range from $80 – $250 per session without insurance.

Ways to reduce the cost:
• Check if your health insurance covers mental health visits (most plans must by law)
• Ask your therapist for a sliding-scale rate — many offer reduced fees
• Community mental health centers often charge on a low-income scale
• Open Path Collective (openpathcollective.org) lists therapists at $30–$80/session
• University training clinics offer low-cost sessions with supervised graduate students
🔍 Find a Therapist 💸 Low-Cost Options

If cost is a barrier right now, start with the free 988 Lifeline above — counselors can also help connect you to affordable local services.
